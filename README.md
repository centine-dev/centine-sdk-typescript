# Centine Node SDK

[![NPM version](https://img.shields.io/npm/v/@centine/sdk.svg)](https://npmjs.org/package/@centine/sdk) ![npm bundle size](https://img.shields.io/bundlephobia/minzip/@centine/sdk)

## Installation

### npm

```sh
npm i @centine/sdk
```

### pnpm 

```sh
pnpm i @centine/sdk
```

### Bun

```sh
bun i @centine/sdk
```

## Usage

```js
import Centine from "@centine/sdk"
```
